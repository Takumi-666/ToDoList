詰まった所
・day3のgo run main.goがうまくいかない
理由はGOPATHがうまくできてない。記入ミス。データベースの接続ミス。だった。
行った対処はGoの再インストール、GOPATHの再設定。コードの修正。
原因　go run main.goを打つディレクトリの場所が違った。開いているVScodeからじゃなくてDocker　Desktopから立ち上げたserverのターミナルに移動してそこでgo run main.goだった。
